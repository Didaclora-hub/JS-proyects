Selectors CSS

Aplicant selectors CSS a diferents parts d'un document HTML

1. El selector d'element selecciona elements basats en el seu nom d'etiqueta HTML.

  1. Els selectors CLASS seleccionen elements basats en le valor de l'atribut CLASS.
  2. Els selectors ID seleccionen elements basats en el valor de l'atribut ID.
  3. Els selectors d'atribut seleccionen basats en el valor de l'atribut.
  4. El selector universal selecciona tots els elements.
